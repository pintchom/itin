# Team standups

One markdown file per **calendar day**, shared by the whole team.

## File naming

```text
docs/standups/YYYY-MM-DD.md
```

Example: `2026-05-24.md`

## Format

Use a section per teammate. Copy [template.md](template.md) when starting a new day.

```markdown
# Standup — 2026-05-24

## Alex

**Yesterday**
- …

**Today**
- …

**Blockers**
- None

## Sam
…
```

## Rules

- Keep bullets short; link PRs or TODO items when useful.
- **Blockers** should name what you need (review, credential, decision).
- Agents: append under the relevant name after a session with meaningful progress.
- Do not delete others' sections; add a new dated file each day.

## Latest

Check the most recent `YYYY-MM-DD.md` in this folder at session start (see [AGENTS.md](../../AGENTS.md)).
